# Car Price Prediction Using Machine Learning

## Objective
To build a machine learning model that predicts the selling price of used cars based on various features such as car age, fuel type, transmission type, kilometers driven, and present market price.

## Steps Performed
1. Imported and explored the dataset.
2. Cleaned and preprocessed the data.
3. Handled categorical variables using encoding techniques.
4. Created a new feature representing the age of the car.
5. Performed Exploratory Data Analysis (EDA).
6. Generated correlation heatmaps and visualizations.
7. Split the dataset into training and testing sets.
8. Trained a Random Forest Regressor model.
9. Evaluated model performance using R² Score and MAE.
10. Predicted car selling prices using the trained model.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Outcome
A machine learning model was successfully developed to predict car selling prices with good accuracy. The project demonstrated how factors such as car age, fuel type, and market price influence resale value.
